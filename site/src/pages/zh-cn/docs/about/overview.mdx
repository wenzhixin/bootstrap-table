## 团队

Bootstrap Table 目前由 GitHub 上的 [wenzhixin](https://github.com/wenzhixin) 维护。我们正在积极扩充团队。如果你对大规模 CSS、编写与维护原生 JavaScript 插件，以及改进前端构建工具流程感兴趣，欢迎与我们联系。

## 参与其中

通过[提交 Issue]([[config:repo]]/issues/new) 或发送 Pull Request 参与 Bootstrap Table 的开发。更多开发信息请参阅我们的[贡献指南]([[config:repo]]/blob/v[[config:currentVersion]]/.github/CONTRIBUTING.md)。
