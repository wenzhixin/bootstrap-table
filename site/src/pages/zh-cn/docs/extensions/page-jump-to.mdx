## 选项

### showJumpTo

- **属性:** `data-show-jump-to`

- **类型:** `Boolean`

- **详情:**

  控制是否启用“跳转到页”功能，可通过 `data-show-jump-to` HTML 属性配置。设置为 `true` 时，将显示页码跳转输入框。

- **默认值:** `false`

### showJumpToByPages

- **属性:** `data-show-jump-to-by-pages`

- **类型:** `Number`

- **详情:**

  设置显示“跳转到页”功能的最小总页数。仅当表格总页数大于或等于该设定值时，才会显示页码跳转功能。

- **默认值:** `0`
