    Cell Input

    Plugin to add input/select element on the cells of table.

    Use Plugin: bootstrap-table-cell-input

    Usage

    Options

    cellInputEnabled

    • type: Boolean

    • Detail:

      Set to true to enable the plugin.

    • Default: false

    Column options

    cellInputEnabled

    • type: Boolean

    • Detail:

      Set to true to make the cell editable.

    • Default: false

    cellInputType

    • type: String

    • Detail:

      Defines the type of input. available options: text and select.

    • Default: text
